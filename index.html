Proyecto Silencio Activo

Por el grupo Peñarol Inteligencia D1

Somos Peñarol Inteliencia D1, un grupo de estudiantes de informática que desarrolla un proyecto educativo para ayudar tanto a alumnos como a profesores de todo el sistema educativo uruguayo para que sus clases se desenvuelvan de una manera más amena y sin pausas.

¿Para que sirve "Silencio Activo"?

“Silencio Activo” sirve para generar un ambiente más adecuado para concentrarse.
El sistema está compuesto por 3 luces LED y un microcontrolador, si el sonido del aula es el adecuado entonces la luz verde estará encendida. Si el sonido del entorno no es el recomendable entonces se prenderá la luz amarilla, y si el ruido es intolerable entonces se prenderá el color rojo.

"Silencio Activo" está especialmente enfocado para ayudar a personas con TDAH para que su aprendizaje y concentración sean mas amenos. Y ayuda a autorregularse sin necesidad de intervención directa por parte del docente
