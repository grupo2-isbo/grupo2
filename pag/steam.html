2. Descripción

  • Esto es un proyecto simple para poder resolver quizas un problema que es frecuente de una forma mas visual para el estudiante y para el profesor el cual este dando la calse.
  • Intentamos resolver un problema muy comun en muchos centros educativos, el ruido, a traves de nuestro proyecto buscamos resolver Esto, para que cada clase o cada hora de estudio sea aprovechada al maximo con las mejores calidades de ambiente posible (respetco al ruido) avisando con luces RGB Rojo para un mal ambiente, amarillo para algo que puede mejorar, y verde para un entorno perfecto de aprendizaje.

3. Materiales

  • En primera instancia ncesitamos: - 2 micro:bit V1 (el cual seria el emisor) y micro:bit v2 -5 cables macho y 4 hembras -2 tarejtas de expancion (en nestro caso usamos solo 1 y una del kit IoT kit que funciona como una tarejta de expancion) -1 protoboard para que el microbit recptor pueda emitir los colores correspondientes. -3 recistencias -1 luz RGB o en su defecto 4 luces las cuales sean de su color correspondinte (rojo, amarillo, verde) pero esto implica un cambio en el codigo ya que el nuestro esta realizado solamente con una luz RGB. -un sensor de sonido, en nuestro caso utilizamos el Noise sensor del IoT Kit.

4. Diseño Mecánico

  • El ensamblaje de nuestro poryecto es algo simple, practico y flexible, ya que una de las partes, (el receptor) esta conectado a la pc para tomar el sonido ambiente del aula, y luego medir el ruido, la otra parte estaria dentro de una caja para protejer el sistema y con el RGB por fuera para mostrar en que nivel de ruido se encuentra el area. El receptor puede estar conectado a la pc como un USB en el dispositivo del profesor que es donde el esta dictando la clase.

6. Diseño Software

  • [Realizar una descripción de como funciona la solución a nivel de software]

  • Descripción del funcionamiento a nivel de software:

    • Se utilizan dos micro:bit, uno de emisor y otro como receptor.

    El sistema utiliza un módulo de radio para la interconexión entre los dos micro:bit.

    El emisor, emitirá vía radio el ruido que se detecta desde el noise sensor.

    Según el nivel de ruido recibido, se controlará el color LED a prender.

  • Este es el codigo del Micro:bit receptor: 
                    from microbit import
                import radio
                
                radio_group = 5
                
                def set_color(r, g, b): #set color in leds
                    pins.analog_write_pin(AnalogPin.P1, r) # P1 is the pin number 1 of the microbit/arduino
                    pins.analog_write_pin(AnalogPin.P2, g)
                    pins.analog_write_pin(AnalogPin.P4, b)
                
                def sColor(color):
                    if color == "Red":
                        set_color(1023, 0, 0) # Red
                    elif color == "Green":
                        set_color(0, 1023, 0) # Green
                    elif color == "Blue":
                        set_color(0, 0, 1023) # Blue
                    elif color == "White":
                        set_color(1023, 1023, 1023) # White
                    else:
                        raise("Invalid Color.")
                
                def radioReceiver(name, ruido):
                    if ruido <= 86:
                        sColor("Green") # Green
                        basic.show_string("G")
                        print("G")
                        basic.pause(300)
                    elif ruido > 86 and ruido <= 171:
                        sColor("Orange") # Orange, like in a traffic light
                        print("O")
                        basic.show_string("O")
                        basic.pause(300)
                    elif ruido > 171:
                        sColor("Red") # Red
                        print("R")
                        basic.show_string("R")
                        basic.pause(300)
                    else:
                        sColor("White") # White
                        print("W")
                        basic.show_string("W")
                        basic.pause(300)
                
                def on_forever():
                    radio.set_group(radio_group)
                    radio.on_received_value(radioReceiver)
                
                basic.forever(on_forever)
